feat: implementação do módulo de gerenciamento de usuários

Implementa sistema completo de autenticação e gerenciamento de usuários
com Spring Boot, Spring Security e JWT.

Funcionalidades implementadas:
- Autenticação JWT com geração e validação de tokens
- CRUD completo de usuários com validações
- Sistema de roles (ADMIN/CLIENTE) e permissões por módulo
- Soft delete com possibilidade de restauração
- Filtros avançados de busca (nome, email, role, status)
- Paginação e ordenação de resultados
- Hash de senhas com BCrypt
- Auditoria automática (data criação, atualização, exclusão)
- Controle de acesso baseado em roles
- Tratamento global de exceções

Estrutura técnica:
- Arquitetura em camadas (Controller/Service/Repository)
- DTOs para requisições e respostas
- JPA Specifications para queries dinâmicas
- Spring Security configurado com filtros JWT
- CORS configurado para integração com frontend

Endpoints implementados:
- POST /api/auth/login - Autenticação
- GET /api/usuarios - Listar todos
- POST /api/usuarios/filtros - Busca com filtros
- POST /api/usuarios - Criar usuário
- GET /api/usuarios/{id} - Buscar por ID
- PUT /api/usuarios/{id} - Atualizar usuário
- PUT /api/usuarios/{id}/permissoes - Atualizar permissões
- DELETE /api/usuarios/{id} - Soft delete
- PUT /api/usuarios/{id}/restaurar - Restaurar usuário
- DELETE /api/usuarios/{id}/permanente - Hard delete

Configurações:
- Suporte a H2 (desenvolvimento) e PostgreSQL (produção)
- Application properties para dev e prod
- JWT configurável via properties
